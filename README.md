# Light Modern 2026

A clean, modern, and carefully tuned light theme for Visual Studio Code.

This theme is designed to reduce visual noise while maintaining strong contrast and readability for long coding sessions.

## ✨ Features

- 🧼 Clean and minimal light UI
- 🎯 Improved focus contrast for editor elements
- 🪶 Soft selection and highlight colors
- 🧠 Reduced visual noise in sidebar and panels
- ⚡ Carefully tuned syntax and UI balance
- 📦 Fully compatible with VS Code theme system

## 🚀 Installation

### From VS Code Marketplace

Search for: `Light Modern 2026`

Or install directly from the Extensions panel.

## ⚙️ Usage

After installation:

1. Open Command Palette
   `Ctrl + K` → `Ctrl + T`
2. Select `Light Modern 2026`

## 📌 Notes

* Designed for readability over saturation
* Optimized for long coding sessions
* UI elements are intentionally softened to reduce fatigue
* Works best in light environments
